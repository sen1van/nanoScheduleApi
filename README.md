Проект отменён за ненадобностью

последние слова:
- вы можете создать\поменять рассписание по примеру 2527.yaml
- программа берёт данные для расспиание из {url}.yaml (доступ апи в формате domain.example/url)
- в timetable можно поменять время занятий
- 0 и 1 в начале урока означают чётную и нечётную неделю (может не соответсвовать реальности, проект умер в зачатке)
- вторая цифра (1;2;3;4;5;6;7) это номер урока в рассписании, чтоб подтянуть время из timetable.yaml
